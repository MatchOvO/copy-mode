# copy-mode
An browser extension allow user to copy the text on web

open in [Microsoft Store](https://microsoftedge.microsoft.com/addons/detail/%E5%A4%8D%E5%88%B6%E6%A8%A1%E5%BC%8F/lobgilnkcbbpldfkgnmhchlidiclangj)

## Usage
* `step1`: Download an install the extension on [Microsoft Store](https://microsoftedge.microsoft.com/addons/detail/%E5%A4%8D%E5%88%B6%E6%A8%A1%E5%BC%8F/lobgilnkcbbpldfkgnmhchlidiclangj)
* `step2`: Turn on the extension ***copy mode***
* `step3`: Click the icon of extension in the address bar on your browser
* `step4`: Turn on the copy mode(复制模式)
